# Svaigas Fishing

A consolidated repository for fishing research, guides, and related projects.

## Subprojects

* **[Carp Feed Research & Guide](https://svaigas.github.io/fishing/carp-feed/)** - Science-backed feeding strategies, optimal protein-to-fat ratios, and temperature-dependent bait guides for catching 1-3kg carp.

## Deployment

The projects in this repository are served as static sites via [GitHub Pages](https://pages.github.com/). 

To set this up:
1. Go to your repository **Settings** on GitHub.
2. Select **Pages** from the sidebar.
3. Under Build and deployment, set **Source** to `Deploy from a branch`.
4. Select the `main` branch and the `/ (root)` folder.
5. Click Save.

Your static files will be instantly deployed and accessible over the internet!
